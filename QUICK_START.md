# ⚡ Quick Start Guide

## 🚀 Get Running in 5 Minutes

### Step 1: Create GitHub Repository (1 min)
1. Go to https://github.com/new
2. Name: `student-performance-app`
3. Click "Create repository"

### Step 2: Push Code (2 min)
```bash
# Clone your new repo
git clone https://github.com/YOUR_USERNAME/student-performance-app.git
cd student-performance-app

# Copy all files from this directory
# Then:
git add .
git commit -m "Initial commit: Student performance app"
git push origin main
```

### Step 3: Watch It Run (2 min)
1. Go to your repo on GitHub
2. Click **Actions** tab
3. Watch the workflow run
4. Download report from **Artifacts**

## ✅ That's It!

Your automated student performance system is now live!
